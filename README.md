# 🌿 Smart AgriAI System

An AI-powered agricultural support system designed to help farmers detect crop diseases, predict risks, and get smart advisory for better decision-making.

---

## 🚀 Features

🔍 **Crop Disease Detection**  
- Upload or capture crop image  
- AI model detects disease or healthy crop  

🌦 **Weather Forecasting & Risk Prediction**  
- Displays upcoming weather conditions  
- Predicts disease risk based on temperature  

🧠 **Smart Advisory System**  
- Provides farming suggestions  
- Supports multiple languages (English, Hindi, Marathi)  

---

##  AI/ML Model Used

- **Convolutional Neural Network (CNN)**
- Model converted to **TensorFlow Lite (.tflite)** for faster performance
- Used for image-based disease classification

---

## ⚙️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **AI Model:** TensorFlow / Keras (CNN)  
- **Deployment:** Streamlit Cloud  
- **Weather API:** Open-Meteo API  

---

## 🎯 Objective

To integrate AI with agriculture and help farmers:
- Detect diseases early  
- Reduce crop loss  
- Improve productivity  
- Promote sustainable farming
